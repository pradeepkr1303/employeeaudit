# employeeaudit

# Prerequisites
1. java - 1.8
2. Maven
3. MySql server

# DB Setup
Follow the queries in dbQueries.txt file

# Steps to run the project
1. Download and extract the file.
2. Open eclipse and select import project from file system.
3. Select the extracted root folder and eclipse will automatically detect the project and will import it.
4. Right click on project name in project explorer.
5. click Run As-->Java Application
